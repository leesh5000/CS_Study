## Sorting

### 1. 버블정렬
1. 첫번째 값과 두번쨰 값을 비교
2. 배열의 끝까지 1번 실행
3. 정렬이 다 될떄까지 1,2번 반복

#### 시간복잡도
- O(n<sup>2<supb>)의 시간복잡도
- 정렬된 배열의 경우는 O(n)

### 2. 선택정렬
1. 최소값을 찾음
2. 해당 최소값을 맨 앞의 값과 교체
3. 맨 앞을 제외한 나머지 데이터들에 대해서 1,2번 반복

#### 시간복잡도
- O(n<sup>2<supb>)의 시간복잡도
